# Цифровой профориентационный навигатор

Это учебный веб-проект по дисциплине **«Обучение служением»**.

Сайт помогает обучающимся 8-11 классов МКОУ «СШ № 14» разобраться в профессиях, образовательных маршрутах и первых шагах для выбора будущего направления.

## Зачем нужен проект

У школьников не всегда есть простой и постоянный доступ к профориентационной информации. Особенно сложно, когда профориентация ограничивается отдельными классными часами, тестами или разовыми встречами.

Этот сайт нужен, чтобы собрать в одном месте:

- понятные описания профессий;
- маршруты обучения после 9 и 11 класса;
- мини-тест для подбора подходящих направлений;
- информацию о профориентационных встречах;
- материалы, которые школа сможет использовать после завершения проекта.

## Что есть на сайте

### Мини-тест

Небольшой тест помогает подобрать профессии по интересам, любимым предметам, формату работы и желаемому пути обучения.

После прохождения сайт показывает несколько подходящих профессий и объясняет, почему они попали в результат.

### Каталог профессий

В каталоге собраны профессии из разных сфер: IT, образование, медицина, дизайн, инженерия, сельское хозяйство, право, бизнес и другие.

Для каждой профессии указаны:

- краткое описание;
- основные задачи;
- школьные предметы;
- нужные навыки;
- первые шаги;
- связанные образовательные маршруты.

### Маршруты обучения

Раздел показывает возможные пути к профессиям:

- после 9 класса;
- после 11 класса;
- через колледж;
- через вуз;
- через курсы;
- через практический проект.

Маршруты можно искать и фильтровать.

### Профориентационные встречи

На сайте есть раздел с планом встреч с представителями разных профессиональных сфер.

Для каждой встречи указаны:

- тема;
- срок;
- формат;
- аудитория;
- предполагаемый спикер;
- материалы;
- связанные профессии;
- ожидаемый результат.

### Информация о проекте

На главной странице также есть краткая информация о команде, заказчике, сроках и этапах проекта.

## Технологии

Проект сделан на стеке:

- Vite;
- React;
- TypeScript;
- Tailwind CSS.

## Как запустить

Установить зависимости:

```bash
npm install
```

Запустить локальный сервер:

```bash
npm run dev
```

Проверить production-сборку:

```bash
npm run build
```

После запуска сайт будет доступен по адресу, который покажет Vite в терминале.
